# How to Fill Forms Automatically with Filleasy

## Step-by-Step Guide

### Step 1: Make Sure You Have Profile Data

Before filling forms, ensure your profile is set up:

1. Click the Filleasy extension icon
2. Click the Settings (‚öôÔ∏è) icon or "Edit" button
3. Fill in your profile information:
   - First Name, Last Name, Email (required)
   - Phone Number
   - Address
   - Other fields as needed
4. Click "Save Profile"

### Step 2: Navigate to a Google Form

1. Open a Google Form in your browser
2. The form URL should look like:
   - `https://docs.google.com/forms/...` 
   - `https://forms.gle/...`
3. Wait for the form to fully load

### Step 3: Open Filleasy Popup

1. Click the Filleasy extension icon in your Chrome toolbar
2. The popup should open
3. You should see "Form Detected" status (if on a Google Form)

### Step 4: Fill the Form

**Option A: Fill Directly**
1. Click "Fill All Fields" button
2. The form will fill automatically
3. Review the filled fields
4. Submit the form

**Option B: Preview First (Recommended)**
1. Click "Preview & Edit" button
2. Review which fields will be filled
3. Check the values
4. Click "Fill Form" to proceed

### Step 5: Review and Submit

1. Check all fields are filled correctly
2. Make manual corrections if needed
3. Submit the form

## Why "Please navigate to a Google Form first" Error Appears

This error appears when:

1. **You're not on a Google Form page**
   - The extension only works on Google Forms
   - Make sure the URL contains `docs.google.com/forms` or `forms.gle`

2. **The form hasn't loaded yet**
   - Wait a few seconds for the page to fully load
   - Refresh the page and try again

3. **You're on the wrong type of form**
   - The extension only works on Google Forms
   - It doesn't work on other form platforms (Microsoft Forms, Typeform, etc.)

4. **URL Check Issue (Technical)**
   - The extension checks if the URL contains specific strings
   - If the URL format is different, it might not detect it

## Troubleshooting

### Error: "Please navigate to a Google Form first"

**Solution:**
1. Make sure you're on a Google Form (check the URL)
2. The URL should contain `docs.google.com/forms` or `forms.gle`
3. Refresh the page
4. Try clicking "Fill All Fields" again

### Form Not Filling

**Possible causes:**
1. Profile data not filled - Go to Settings and fill your profile
2. Fields don't match - Some form fields might not match your profile fields
3. Form is read-only - Some forms don't allow auto-filling
4. Form hasn't loaded - Wait for the form to fully load

**Solution:**
1. Use "Preview & Edit" to see which fields will be filled
2. Check your profile has the required data
3. Try refreshing the page
4. Manually fill fields that didn't auto-fill

### Extension Icon Not Showing

**Solution:**
1. Click the Extensions icon (puzzle piece üß©) in Chrome toolbar
2. Find "Filleasy"
3. Click the pin icon (üìå) to pin it
4. Icon should appear in toolbar

## Tips for Best Results

1. **Complete Your Profile**: More complete profile = better field matching
2. **Use Preview First**: Always preview before filling important forms
3. **Check Field Names**: Form field names should match common patterns (First Name, Email, etc.)
4. **Manual Corrections**: Some fields may need manual adjustment
5. **Multi-page Forms**: Fill each page separately

# Icons Directory

Please add the following icon files to this directory:

- `icon16.png` - 16x16 pixels (for toolbar)
- `icon48.png` - 48x48 pixels (for extension management)
- `icon128.png` - 128x128 pixels (for Chrome Web Store)

## Icon Design Guidelines

The icon should represent form filling and automation. Suggested design:
- Simple, recognizable symbol (e.g., document with checkmark, form with arrow)
- Use brand color #6366F1 (indigo)
- Clean, modern design
- High contrast for visibility at small sizes

You can create these icons using:
- Online icon generators
- Design tools (Figma, Adobe Illustrator)
- Icon libraries (Font Awesome, Material Icons)
